# New Game! - Para Devs UwU

## ¿Qué es esto? >:3

New Game! es un manga sobre Aoba-chan, una dev junior que entra a Eagle Jump (estudio de videojuegos) y vive todas las aventuras típicas de la industria tech OwO

## ¿Por qué debería importarme? 7w7

### Workplace Vibes
- **Equipos diversos**: Como en cualquier startup kawaii
- **Mentoring**: Senpais ayudando a kouhais (senior/junior dynamics)
- **Crunch time**: Esas noches de deploy que todos conocemos >:<

### Technical Stuff
```
Game Dev Pipeline uwu:
├── Concept & Design  
├── Programming  
├── Assets & Testing  
└── Release (and crying)
```

### Real Developer Problems
- **Impostor syndrome**: "No sé nada" mood 24/7
- **Learning curve**: Siempre hay algo nuevo que aprender unu
- **Team dynamics**: Diferentes personalidades = drama pero también magic
- **Work-life balance**: ¿Qué es eso? 7w7

## ¿Para quién es?

**Junior Devs**: Te vas a sentir identificado con Aoba struggling con todo >:3

**Senior Devs**: Nostalgia de tus primeros días + reflexiones sobre mentoring uwu

## Fun Facts OwO
- Basado en experiencias reales de la industria gaming japonesa
- Tiene anime de 2 temporadas para los que prefieren no leer
- Es "cute girls doing tech things" pero con problemas laborales reales

## Conclusión

New Game! es básicamente "The Office" pero con waifus y videojuegos. Muestra la realidad del desarrollo de software: el estrés, la colaboración, el growth, y sí... también las crisis existenciales a las 3am debuggeando código unu

**TL;DR**: Manga kawaii que actually enseña sobre team dynamics y cultura tech. Perfect para procrastinar mientras aprendes algo útil 7w7
